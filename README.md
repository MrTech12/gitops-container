# GitOps-Container

## Overview
This project is for creating a Docker container, which provides access to popular GitOps CLI tools.

## Motivation


## Bundled tools
- curl
- nano
- tmux
- kubectl
- helm
- ArgoCD CLI
- Argo Rollouts kubectl plugin
- Terraform CLI
- DigitalOcean CLI (doctl)
- Pulumi CLI
- Dapr CLI
